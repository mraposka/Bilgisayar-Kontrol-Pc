PCKontrol N0Kayıp adlı projem, bilgisayar sayısının fazla olduğu kuruluşlarda geceleri açık kalan bilgisayarların kontrolünü sağlamak için tasarlanmıştır. Programı kurduktan sonra web site kısmının kurulumunu yapıp projenizin çalışmasını web sitenizden izleyebilirsiniz.


Kurulum için aşağıdaki adımları izleyiniz:

1-BilgisayarAcKapaUygulamasi klasöründeki projeyi açıp 90. satırdaki "site" değişkenini sitenizin adı ile değiştirin.(örnek domain=abcdf.com site değişkeni=abcdfg)
2-Update klasöründeki projeyi açıp 180. satırdakii ftp server adresini ve 193. satırdaki ftp kullanıcı adı ve şifre yerlerini değiştirin.(örnek ftp server adresi:ftp.enesvekadir.com/ ve örnek ftp kullanıcı adı ve şifre:"enesvekadir", "enes++52" olarak belirtilmiştir.)
3-BilgisayarAcKapaUygulamasi ve Update projelerini derleyip .exe hallerini Setup klasöründe KurulumKlasör adlı dizine kopyalayıp değiştirin.
4-Setup klasöründeki .NET FrameWork 4.6.2 dosyasını kurun.
5-Install dosyasına tıklayarak hızlı bir kurulum yapabilirsiniz.

Web sitesi kısmını kurmayı unutmayınız. Web sitesi projemi Bilgisayar-Kontrol-Web adlı repomda bulabilirsiniz.
